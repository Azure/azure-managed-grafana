# Support

## How to file issues and get help

This repository uses GitHub Issues to track Azure Managed Grafana feedback, bug reports, feature requests, and questions. Before opening a new issue, search the existing issues to avoid duplicates.

For new bugs, feature requests, or product feedback, open an issue at:

- https://aka.ms/managed-grafana/issues

## Microsoft Support Policy

Community support for this repository is provided through GitHub Issues. For production-impacting Azure Managed Grafana service issues, use your existing Azure support plan through the Azure portal.
